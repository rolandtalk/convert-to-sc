# Roadmap

## Milestone 1: GitHub-Ready Foundation

- reframe the repo around CSC
- document product scope
- add GitHub templates
- clean repo hygiene for local artifacts

## Milestone 2: Symbol Extraction

- add a parser that extracts likely tickers from free-form text
- support manual review and correction
- persist original text and parsed symbols

## Milestone 3: Screenshot Automation

- generate chart URLs from symbols
- automate browser navigation
- capture and store screenshots
- handle failures and retries

## Milestone 4: Web Publishing

- gallery UI for screenshot browsing
- filter by run, symbol, and date
- expose API endpoints for screenshot records

## Milestone 5: Production Readiness

- move storage beyond SQLite if needed
- add tests for parsing and screenshot jobs
- add deployment and monitoring improvements
